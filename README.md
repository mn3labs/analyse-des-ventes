# Analyse des Ventes 2023-2024

> Projet Data Analyst — Exploration, nettoyage et visualisation de 200 transactions commerciales sur 2 années.

---

## Apercu du projet

| Metrique | Valeur |
|---|---|
| CA Total Livre | 322 336 € |
| Transactions | 200 |
| Vendeurs | 3 |
| Categories | 2 (Informatique, Telephonie) |
| Regions | 3 |
| Marge Moyenne | 30,2 % |
| Taux d'Annulation | 13,5 % |

---

## Objectifs

- Identifier les **tendances de ventes** par periode, region et canal
- Comparer la **performance des vendeurs**
- Analyser la **rentabilite par categorie** de produit
- Mettre en evidence les leviers d'optimisation du **taux de conversion**

## Competences demontrees

- **Excel avance** : tableaux croises dynamiques, formules RECHERCHEV / INDEX-EQUIV, mise en forme conditionnelle
- **Analyse de donnees** : segmentation, KPIs, analyse de la marge et du taux d'annulation
- **Data visualisation** : graphiques dynamiques, dashboard interactif
- **Esprit analytique** : interpretation des resultats, formulation de recommandations metier

---

## Insights cles

1. **L'informatique represente 66 % du CA** — priorite strategique confirmee
2. **Bruno Lefevre surperforme** de 14 % vs Alice Martin — bonnes pratiques a partager
3. **13,5 % de taux d'annulation** — marge d'amelioration sur la gestion des commandes
4. **La boutique physique depasse l'e-commerce** (53 % vs 47 %) malgre la tendance digitale
5. **Ile-de-France et ARA presque a egalite** — potentiel de croissance en Nouvelle-Aquitaine
